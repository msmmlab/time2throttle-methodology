# Time2Throttle — Test Methodology

This repository is the public source of truth for the [Time2Throttle](https://time2throttle.com) test methodology.

## Purpose

Time2Throttle is an independent lab measuring thermal throttling, surface temperature, and real battery runtime in smart glasses and wearables — under real workloads, with published methodology, starting from devices bought at retail.

This methodology is published **before any results**, by design. Reproducibility is the product.

## Critique

If you build, deploy, or test wearables and see a flaw — in the workloads, the mounting, the metrics, or the statistics — **[open an issue](https://github.com/msmmlab/time2throttle-methodology/issues)**. Credited corrections enter the changelog.

## Versioning

This methodology changes only by public versioned revision with changelog — never retroactively.
